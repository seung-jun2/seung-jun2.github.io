당신 마켓(팀프로젝트)

프로잭트 개요

당신마켓은 중고 신발 거래를 위한 온라인 플랫폼입니다. 사용자들은 안전하고 신뢰할 수 있는 환경에서 중고 신발을 구매하고 판매할 수 있습니다. 우리의 목표는 사용자들이 쉽고 빠르게 원하는 제품을 찾고 거래할 수 있도록 돕는 것입니다. 함께하면 더 편리한 중고 신발 거래 경험을 만들어나갈 수 있습니다.

사용 기술:

  • 언어: Java
  • 프레임워크: MyBatis
  • 데이터베이스: Oracle Database
  • HTML/CSS/JavaScript

기획 및 기능:

  • 공지사항 CRUD 기능 구현: MyBatis를 활용하여 공지사항의 생성, 조회, 수정, 삭제 기능을 개발했습니다.
  • 데이터베이스 설계 및 구축: Oracle Database를 활용하여 프로젝트의 데이터베이스를 설계하고 구축했습니다.
  • 개발 환경 구성 및 개발 진행: 이클립스 IDE를 사용하여 효율적인 개발 환경을 구성하고 프로젝트의 개발을 진행했습니다.

TeamPlus(팀프로젝트)

프로잭트 개요

TeamPlus는 사람들 간의 협업을 촉진하기 위해 시작되었습니다. 이 플랫폼은 사용자들이 효율적으로 팀을 구성하고 프로젝트를 관리할 수 있도록 돕습니다. 사용자들이 아이디어를 공유하고 의견을 나누며, 함께 일하는 과정을 원활하게 만들기 위해 이 프로젝트를 개발했습니다. 함께하면 보다 효율적인 협업 경험을 만들어나갈 수 있습니다.

이 프로젝트에서 사용된 기술 스택은 다음과 같습니다:

  • 언어: Java
  • 데이터베이스: Oracle
  • IDE: IntelliJ
  • 프레임워크: Spring Boot
  • 라이브러리: Gson, jQuery

기획 및 기능:

  • SNS 로그인 API: 사용자가 소셜 미디어 계정을 통해 로그인할 수 있도록 구현했습니다.
  • 사용자 CRUD(Create, Read, Update, Delete) 기능: 사용자 데이터를 생성, 조회, 수정, 삭제할 수 있는 기능을 담당했습니다.

Bookey(팀프로젝트)

프로잭트 개요

Bookey는 독자들에게 다양한 책을 소개하고 추천해주는 플랫폼입니다. 이 프로젝트는 독서를 즐기는 사람들에게 다양한 책을 발견하고 탐색할 수 있는 기회를 제공합니다. 우리는 독자들이 다양한 장르와 주제의 책을 찾을 수 있도록 다양한 추천 알고리즘을 개발하고, 독자들끼리 책에 대한 리뷰와 의견을 공유할 수 있는 커뮤니티를 구축했습니다. 함께하면 보다 풍부한 독서 경험을 만들어갈 수 있습니다.

이 프로젝트에서 사용된 기술 스택은 다음과 같습니다:

  • 언어: Java
  • 프레임워크: Spring
  • 데이터베이스: MariaDB
  • 개발도구: IntelliJ IDEA

협업은 GitHub을 통해 코드 관리를 하였고, API 문서화는 Swagger를 이용했습니다.

기획 및 기능 :

  • 검색 고도화 기능: ChatGptAsssistantAPI를 활용하여 구현하였습니다.
  • 댓글 CRUD 기능
  • S3 서버 이미지 업로드: 사용자가 업로드한 이미지를 저장하고 관리하기 위해 Amazon S3 서버를 사용했습니다.
  • 인터파크도서 사이트 랭킹 데이터 크롤링
